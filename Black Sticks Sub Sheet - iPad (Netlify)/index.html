BLACK STICKS — SUB SHEET

GAME-DAY GUIDE

1. Before the game (on the Mac or iPad)
• Pick the game in the top bar (or New for a fresh one — squad carries over).
• For each quarter tab, pick the structure (e.g. 6 @ BACK – 5 MF – 5 STR). It sets how many of each line must be on. "Apply to ALL" if you run one shape all game.
• Rotation units (your "take the same person off" system): open the Rotation units panel, make a unit per position group — starters first, interchange last — set 1st sub & cadence (e.g. 1st sub 4', every 4' = your 4-on-4-off), tick Q1–Q3, hit GENERATE ALL. The grid paints itself. Leave Q4 unticked and paint it by hand for minute control.
• Fine-tune anywhere by painting: click/tap a cell = on/off, drag to paint a block (Mac). Right-click a cell to set an exact position or blank.
• GAME tab: set target minutes for key players (Woods 50, etc.). Paint until the +/- column is all green ticks and Quarter Checks says all four quarters are legal.
• Hit Export — that file is your backup. Print if you want paper as a fallback.

2. At the game (iPad on the bench)
• Open the quarter tab you're playing. Press ▶ START on the clock when the umpire starts play.
• The pink column follows the match clock. Bright rows = should be ON. Dimmed rows = off. The banner reads out NOW and NEXT subs — just follow the banner.
• Time called off (injury, video referral)? Press ❚❚ STOP. Press START when play resumes.
• Clock drifted from the scoreboard? Use −10s / +10s, or tap the right minute number in the header to jump the clock there.
• Quarter over: press RESET, switch to the next quarter tab, START again.

3. Controlling Q4 minutes live
• Plan Q4 with your finishers loaded (Lane, Russell, Woods on big minutes). If the game state changes, just repaint Q4 cells on the fly — totals, banner and rows update instantly. STOP the clock while you adjust if you need a beat.

4. Getting this onto the iPad
• This file lives in your Dropbox — but the Dropbox/Files preview won't run it. Install the free Documents by Readdle app, connect Dropbox inside it (or AirDrop the file in), and open the file there — it runs fully, saves included.
• Note: iPad saves live on the iPad, Mac saves on the Mac. Move plans between them with Export / Import.
• On iPad: tap = toggle a cell, long-press = the position menu.
Click a cell to put a player on/off for that minute. Click & drag to paint a block. Brush:  |  Right-click any cell to set that one box yourself (any position, blank, or off).  |  Columns count down 15 → 1 (time remaining in quarter).
Rule: every line carries its number for every minute — 3 strikers, 3 midfield, 4 backs, 1 GK (editable in the count rows if the structure changes). Each count row turns green when that minute is right, red when it isn't. The end-of-row total (e.g. 45/45) checks the line's minutes add up to full coverage. Qtr = minutes this quarter, Game = total across all four quarters. The sub plan below builds itself from the grid — that's the sheet the bench runs off.
